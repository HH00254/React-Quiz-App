# React Quiz App

An interactive quiz application built with React and Vite that challenges users with timed questions, tracks answers in real time, and provides a detailed summary at the end of the quiz.

The application focuses on reusable React components, state management, conditional rendering, and user interaction patterns while delivering a polished and responsive UI experience.

---

## Features

* Timed questions with countdown progress bar
* Dynamic answer tracking
* Automatic progression through quiz questions
* Correct / incorrect answer validation
* Skipped question handling
* Quiz summary screen
* Reusable React component architecture
* Responsive and modern UI
* Built using Vite for fast development and performance

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/react-quiz-app.git
```

Navigate into the project:

```bash
cd react-quiz-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
